# 🎨 Arte Digital

Repositório dedicado aos projetos, atividades e materiais desenvolvidos na disciplina de **Arte Digital**. Aqui estão reunidos trabalhos práticos produzidos utilizando ferramentas de desenho e criação digital.

## 🎯 Sobre a disciplina

A matéria de Arte Digital tem como objetivo desenvolver competências relacionadas a:

- Técnicas de desenho e ilustração digital
- Composição visual, cores e texturas
- Uso de ferramentas profissionais de criação gráfica
- Desenvolvimento de criatividade e expressão artística através de meios digitais
- Edição e manipulação de imagens

## 🛠️ Ferramentas utilizadas

![GIMP](https://img.shields.io/badge/GIMP-657D8B?style=flat&logo=gimp&logoColor=white)

Os trabalhos deste repositório foram produzidos com o **GIMP** (GNU Image Manipulation Program), uma ferramenta de código aberto para edição e criação de imagens digitais.
